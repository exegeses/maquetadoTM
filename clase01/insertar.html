Insertar CSS en HTML

Se puede insertar CSS en línea

El CSS en línea se interta utilizando el atributo style.

Se puede insertar CSS de manera local (dentro del HTML)

El CSS se puede también insertar dentro del HTML, dentro del elemento head utilizando el elemento style.
Ver en el encabezado del documento.

Enlace de hoja de estilos (archivo externo)

La manera más práctica de insertar CSS es utilizando un archivo de hoja de estilos.
Este archivo se enlaza con el elemento link y su atributo href
