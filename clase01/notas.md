> URL del repositorio: 
> https://github.com/exegeses/maquetadoTM

## ¿Que significa maquetado?

Maquetado es el proceso de volcar 
a código HTML y CSS el proyecto web que vayamos a realizar

## Fórmula del HTML

    <elemento atributo = "valor">
            objeto
    </elemento>
    
## Fórmula del CSS

    selector {
        atributo: valor;
    }
